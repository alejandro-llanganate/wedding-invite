# Bienvenidos al Amazon - Next.js + Three.js

Un proyecto interactivo inspirado en National Geographic "Bienvenidos al Amazon" con scroll vertical y efectos 3D usando Three.js.

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

## Tecnologías

- Next.js 14
- React 18
- Three.js
- React Three Fiber
- React Three Drei
- Framer Motion
